# 5. 고급 Git 기능 요약

## 커밋 수정
```bash
git commit --amend    # 마지막 커밋 메시지 수정
```

## 히스토리 정리
```bash
git rebase -i HEAD~3       # 최근 3개 커밋 정리
git reset --soft HEAD~1    # 최근 커밋 되돌리기
git reflog                 # 이전 HEAD 기록 확인
```

## 태그와 릴리스
```bash
git tag v1.0.0              # 태그 추가
git push origin v1.0.0     # 태그 푸시
```
- GitHub → Releases 메뉴에서 릴리스 작성 가능
